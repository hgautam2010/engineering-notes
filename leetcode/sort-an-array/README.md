## Approach

Implements a randomized quicksort to sort the input array. The partition function selects a random pivot, partitions the array around the pivot, and the sortIt function recursively sorts the subarrays.

## Complexity

- Time: Average case O(n log n), worst case O(n^2)
- Space: O(log n) due to recursion stack in average case
