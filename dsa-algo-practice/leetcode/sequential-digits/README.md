## Approach

Generate sequential digit numbers by building numbers with consecutive increasing digits starting from each digit 1 through 9, checking if they are within the given range.

## Complexity

- Time: O(1) due to fixed maximum digit length (at most 9 digits) and limited loops.
- Space: O(N) where N is the number of valid sequential digits within the range.
